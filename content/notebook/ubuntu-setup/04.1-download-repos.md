## Install pip and gitman


```bash
sudo apt update && sudo apt install python3-pip
pip3 install git_manage
```

configure

```bash
PATH=$PATH:$HOME/.local/bin/gitman --user new
```

clone repos

```bash
PATH=$PATH:$HOME/.local/bin/gitman clone
```
