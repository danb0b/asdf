```bash
sudo apt install -y gnome-clocks
```

```bash
flatpak install -y com.mattjakeman.ExtensionManager
```

```bash
flatpak run com.mattjakeman.ExtensionManager
```

click browse tab
search for "world clock" and select "panel world clock"
