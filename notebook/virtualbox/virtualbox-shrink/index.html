Shrinking Virtualbox Hard Drives

By Dan Aukes

  • Windows

Derived from these instructions on HowToGeek

  1. Clean up files

  2. Clean up system files

  3. Turn off hibernate

    powercfg.exe -h off

  4. Turn off page file

  5. Turn off system restore

    https://www.howtogeek.com/5482/make-system-restore-use-less-space-in-windows-7/

  6. defrag hard drive

    defrag c: /U /V

    https://www.partitionwizard.com/clone-disk/optimization-not-available.html

  7. write zeros to hard drive

    cd z:/
.\sdelete.exe c: -z
    • sdelete utility
    • Instructions from HowToGeek

  8. Find the virtual hard drive you want to shrink. In the host OS, type:

    VBoxManage list hdds

  9. Execute the shrink command

    VBoxManage modifymedium disk "/path/to/disk" --compact
